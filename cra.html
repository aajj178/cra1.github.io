Frequently Asked Questions

Find answers to common questions about our code review assistant.

What programming languages do you support?

We currently support JavaScript, TypeScript, Python, Java, C#, C++, PHP, Ruby, and Go. We're continuously adding support for more languages based on user demand.

How does the code analysis work?

Our system uses static code analysis combined with machine learning to identify patterns, potential bugs, security vulnerabilities, and areas for improvement. The analysis is based on industry best practices and common pitfalls.

Is my code stored or used for training?

No, we don't store your code or use it for training our models unless you explicitly opt-in to share anonymized snippets to help improve our service. Your code privacy is our top priority.

Can I use this for commercial projects?

Absolutely! All our plans can be used for both personal and commercial projects. The Team plan is specifically designed for professional and enterprise use.

How do I cancel my subscription?

You can cancel anytime from your account settings. There are no cancellation fees, and you'll continue to have access until the end of your billing period.
